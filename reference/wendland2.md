# Wendland 2 compactly supported covariance function

Evaluates the Wendland order-2 (C4) function: for d in (0, 1), C(d) =
(1 - d)^6 \* (35d^2 + 18d + 3) / 3. Returns 0 for d \> 1.

## Usage

``` r
wendland2(d)
```

## Arguments

- d:

  Non-negative distance (scaled by range parameter).

## Value

Covariance value.
