# GEV quantile function (return level)

Computes the r-year return level from GEV parameters.

## Usage

``` r
gev_return_level(mu, sigma, xi, r)
```

## Arguments

- mu:

  Location parameter.

- sigma:

  Scale parameter (NOT log scale).

- xi:

  Shape parameter.

- r:

  Return period in years.

## Value

The r-year return level.
