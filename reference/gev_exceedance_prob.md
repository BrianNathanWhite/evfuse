# GEV exceedance probability

Computes P(Z \> z0) for Z ~ GEV(mu, sigma, xi).

## Usage

``` r
gev_exceedance_prob(z0, mu, sigma, xi)
```

## Arguments

- z0:

  Threshold value.

- mu:

  Location parameter.

- sigma:

  Scale parameter (NOT log scale).

- xi:

  Shape parameter.

## Value

Exceedance probability.
